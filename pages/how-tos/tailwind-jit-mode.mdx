# Use Tailwind JIT (Just-in-time) Mode

1. Enable JIT mode by adding the `mode: "jit"` config inside `tailwind.config.js`.

<CodeSnippet language="js">
  {() => `// tailwind.config.js
module.exports = {
  mode: 'jit',
  purge: [
    // ...
  ],
  theme: {
    // ...
  }
    // ...
}`}

</CodeSnippet>
